PROJETOS DE CIÊNCIA DE DADOS

Construção de soluções para os problemas de negócio, próximo dos desafios reais de uma empresa, utilizando dados públicos de competições de ciência de dados, onde é abordado desde a questão de negócio da empresa até a publicação do modelo, devidamente treinado, em produção, utilizando ferramentas de cloud computing.

Insiders Clustering

Um projeto, em desenvolvimento, de clusterização, onde o problema de negócio da empressa de E-commerce envolve em criar um programa de fidelidade para os seus clientes valiosos, portanto esse projeto tem o objetivo de identificar quem são os clientes elegíveis para esse programa.

O resultado dessa solução, em detalhes, você pode conferir no Github.

As ferramentas utilizadas foram:

  • Git, Github
  • Python, Pandas, Seaborn e Matplotlib
  • Jupyter Notebook, VSCode
  • K-Means, Hierarquical Clustering, DBScan
  • AWS Cloud (EC2, S3, Postgre, SQLite)
  • Metabase Vizualization
  • Visualize on Github

Health Insurance Cross Sell

Um projeto de classificação do tipo “learning to rank”, onde o problema de negócio da empresa de seguro de carro envolve em descobrir quais são os clientes que tem maior propensão de adquirir um segundo produto que está sendo ofertado pela empresa, o de seguro de vida.

O resultado dessa solução, em detalhes de retornos financeiros, você pode conferir no Github.

As ferramentas utilizadas foram:

  • Git, Github
  • Python, Pandas, Seaborn, Matplotlib, Scipy, SKLearn
  • Jupyter Notebook, VSCode, Google Colab
  • Random Forest, Decision Tree, XGboost, KNN, GaussianNB
  • Visualize on Github

Rossmann Sales Prediction

Um projeto de regressão, onde o problema de negócio da empressa de rede de farmácia envolve em saber qual será o retorno financeiro nas próximas seis semanas de cada loja da empresa, afim de ter uma dimensão de qual será o limite de orçamento para uma futura reforma de cada loja.

O resultado dessa solução, em detalhes de retornos financeiros, você pode conferir no Github.

As ferramentas utilizadas foram:

  • Git, Github
  • Python, Pandas, Seaborn, Matplotlib, Scipy, SKLearn
  • Jupyter Notebook, VSCode, Google Colab
  • Random Forest, XGBoost
  • Visualize on Github

PCR Covid Test - NIR

Um projeto de classificação, onde o problema de negócio da empressa de rede de farmácia envolve em classificar os testes positivos e negativos para a Covid de cada amostra utilizando um método chamado Espectroscopia do Infravermelho Próximo (NIR).
